# Pôster Científico

Esta pasta é destinada a armazenar todos os materiais relacionados ao pôster da pesquisa.

**O que colocar aqui:**
- Rascunhos de texto para o pôster.
- Arquivos de design ou links (se estiverem usando Canva, por exemplo, podem criar um arquivo de texto com o link).
- A versão final exportada do pôster (ex: `poster_final.pdf` ou `poster_final.png`).
