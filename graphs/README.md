# Gráficos e Visualizações

Salve aqui os gráficos finais gerados nas análises estatísticas (formatos recomendados: `.png`, `.svg` ou `.jpg`). 

Essas imagens serão utilizadas na construção do pôster e do relatório final.

**Padrão de nomenclatura sugerido:**
`[nome_do_membro]_[descricao_curta].[extensao]`

**Exemplos:**
- `gabriel_success_rate_per_year.png`
- `kaio_reused_vs_virgin_success.png`
- `izac_booster_experience_scatter.svg`
- `jonatas_rocket_family_bar.png`
